# Portfólio Pessoal com Painel Administrativo

Este é um projeto de portfólio pessoal desenvolvido com HTML, CSS, Bootstrap e JavaScript, que inclui um painel administrativo para facilitar a atualização de conteúdo.

## Estrutura do Projeto

- `index.html` - Página principal do portfólio
- `admin.html` - Painel administrativo para edição de conteúdo
- `css/` - Pasta contendo os arquivos de estilo
  - `style.css` - Estilos para a página principal
  - `admin.css` - Estilos para o painel administrativo
- `js/` - Pasta contendo os arquivos JavaScript
  - `main.js` - Funcionalidades da página principal
  - `admin.js` - Funcionalidades do painel administrativo
- `assets/` - Pasta para imagens e outros recursos

## Como Usar

1. Abra o arquivo `index.html` em seu navegador para visualizar o portfólio
2. Para editar o conteúdo, acesse `admin.html` ou clique no link "Admin" na navbar
3. No painel administrativo, você pode:
   - Alterar a foto de perfil
   - Modificar textos (título, subtítulo, sobre, etc.)
   - Adicionar/rem
